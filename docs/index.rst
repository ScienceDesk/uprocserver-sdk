Welcome to Micro Process Server Python (upserver) SDK documentation!
======================================================

Introduction
------------
This SDK contains a Micro Process Server client as well as some helper that can be imported and used in the fuctions to be run by upserver.

Install
-------
Latest release

.. code-block:: bash

   pip install uprocserver-sdk


Development release candidates

Check out for versions `Here <https://pypi.debonzi.dev/#/package/uprocserver-sdk>`_

.. code-block:: bash

   pip install -i https://pypi.debonzi.dev/simple/ uprocserver-sdk==0.1.0+rc.0
